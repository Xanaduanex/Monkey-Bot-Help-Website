Monkey Bot Commands

Key
  • [     ] : Optional Argument
  • {     } : Required Argument
  • * : Moderators Only

Usage: ~bible
Description: Get a random Bible quote.

Usage: ~fact
Description: Generates a random fun fact.

Usage: ~facttoggle
Description: Toggles automatic fun facts on/off

Usage: ~guess
Description: Play a simple guessing game.

Usage: ~help
Description: DM's user website listing all commands.

Usage: ~kick [@user]
Description: Kicks the specified user.

Usage: ~league {-link/-unlink} [League username/@User]
Description: Shows the Ranked Solo/Duo information of the specified League username.

-link: Associates a League username with your Discord ID
-unlink: Removes the League username associated with your Discord ID

Usage: ~pfp {@user}
Description: Shows the profile picture of the specified user. Shows your profile picture if no user is specified.

Usage: ~profile {@user}
Description: Shows the account profile of the specified user. Shows your account profile if no user is specified.

Usage: ~say [message]
Description: Makes Monkey Bot say the provided message.

Usage: ~setmain [#channel]
Description: Sets the channel mentioned to be the channel that the bot sends automatic messages to.

Usage: ~sneakysay [message]
Description: Same thing as ~say, but the command message is deleted for maximum stealth.

Usage: ~xp {@user}
Description: Shows the level and xp of the specified user. Shows your level and xp if no user is specified.
